# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in this project, please **do not open a public issue**.

Instead, report it privately by contacting the maintainer:

* Email: [lar000ki@gmail.com](mailto:lar000ki@gmail.com)

Please include:

* A description of the vulnerability
* Steps to reproduce
* Potential impact
* Any suggested fixes (if available)

## Response process

We will acknowledge your report within a reasonable time and investigate the issue.

If the vulnerability is confirmed:

* A fix will be developed as soon as possible
* A new release will be published
* The reporter may be credited (if desired)

## Supported versions

Only the latest release is actively supported with security fixes.
